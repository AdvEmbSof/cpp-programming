The code available in this repository is a companion of the TSM_AdvEmbSof and
 MA_EmbReal lectures at HES-SO. The description of code snippets is available
under https://embreal.isc.heia-fr.ch/programming/intro.

Cloning the repository at the root of your workspace enables you to build the
different applications as standard {{zephyr_os}} applications and run them on
the selected target device.
